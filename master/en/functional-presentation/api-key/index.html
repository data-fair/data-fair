Clés d’API

Toutes les fonctionnalités de Data Fair sont exposées aux niveaux de l'API. Celle ci peut être appelée en dehors du portail, mais pour les accès restreints il faut passer par l'utilisation d'une clé d'API. Les différentes utilisations possibles de l'API sont décrites plus en détails dans cette section.

Lors de l’ajout d’une clé d’api, il est possible de restreindre l’accès à une seule fonction. Il est ensuite possible de restreindre l’accès à une IP ou à un nom de domaine précis.
